TCP Port Scanner
I developed this project as a way to learn more about how low level networking actually works. Instead of relying on existing tools like Nmap, i wanted to create my own version of it so that i could understand its mechanisms, as well as to get in the habit of building my own tools for when the available ones are not good enough. This was built entirely in Python only using the standard libraries.

⚠️ Disclaimer
This tool is for educational purposes only. You must only scan systems you own or have explicit permission to scan. 

Features
1. Scans a range of TCP ports on a target host as specified by the user.
2. Prints a summary after the scan.


Setup for Windows
1. Clone the repo and cd into it:
git clone https://github.com/cesarcastro33/port-scanner.git
cd port-scanner

2. Create virtual environment
python -m venv venv

3. Activate virtual environment
venv\Scripts\activate

4. Install dependencies
pip install -r requirements.txt
